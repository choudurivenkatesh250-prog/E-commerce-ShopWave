# E-Commerce Web App (ReactJS)

## Description

This is a simple E-commerce web application built using **ReactJS**.
Users can browse products, view details, add items to the cart, and perform a mock checkout.

---

## Features

* Product listing
* Product details page
* Add to cart
* Update/remove cart items
* Total price calculation
* Mock checkout
* Responsive design

---

## API

Data is fetched from **Platzi Fake Store API**.

---

## Tech Stack

* ReactJS
* JavaScript
* CSS
* React Hooks (`useState`, `useEffect`)

---

## Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/your-username/ecommerce-app.git
```

### 2. Go to project folder

```bash
cd ecommerce-app
```

### 3. Install dependencies

```bash
npm install
```

### 4. Start the development server

```bash
npm start
```

### 5. Open in browser

```
http://localhost:3000
```
